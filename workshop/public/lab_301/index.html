Lab 01 - Getting Started with Spring Boot

The completed code for this lab can be found at $COURSE_HOME/spring-boot-basics/final/springbox-catalog.

We’ll begin the first subproject of our complete cloud-native application architecture: SpringBox, which implements the movie browsing portion of a ``Redbox-like'' website. This microservice will provide the Movie Catalog Service, which will allow us to browse movies by genre, browse an individual movie’s details, and search for movies.

We’ll be using the MovieLens dataset, and we’ll model it’s Movie and Genre concepts in this microservice.

Approximate time: 10 minutes

Exercises

Generate a project

  1. In your browser, visit http://start.spring.io.

  2. Fill out the Project metadata fields as follows:

    Generate

    Maven Project

    Spring Boot

    1.4.2

    Group

    lab

    Artifact

    springbox-catalog

    Dependencies

    Web,JPA,Actuator,DevTools,Rest Repositories,H2,Config Client,Eureka Discovery

  3. Click the Generate Project button. Your browser will download a zip file.

  4. On a terminal window,
    Unpack that zip file into $COURSE_HOME/spring-boot-basics/initial.

Add a Rest end point

  1. cd to the directory $COURSE_HOME/spring-boot-basics/initial/springbox-catalog.

  2. Import the project’s pom.xml into your editor/IDE of choice.

  3. Create a class lab.MovieController

  4. Add a @RestController annotation to the class lab.MovieController.

  5. Add the following code to the class lab.MovieController:

    	@Value("${greeting}")
	private String greeting;

	@RequestMapping("/")
	public String greet() {
		return greeting;
	}

Add Greeting

  1. cd to the directory $COURSE_HOME/spring-boot-basics/initial/springbox-catalog.

  2. Import the project’s pom.xml into your editor/IDE of choice.

  3. Rename src/main/resources/application.properties to application.yml

  4. Add the following code to the class application.yml:

    security:
  basic:
    enabled: false # <---  turn off basic security (on by default)
greeting: Hello World

Test the Hello World Endpoint

  1. Build and run the application:

    $ mvn spring-boot:run

  2. Try out the following endpoint. The output is omitted here:

    http://localhost:8080

    Says Hello, World

Clean Up

NOTE

The next step is optional. You can continue the next lab using the workspace that you have created, or clean up and follow the next lab from scratch.

Let’s clean up our application so that we can have a clean start for the next lab

  1. Delete the project from your IDE

Congratulations! You’ve just completed your first Spring Boot application.
